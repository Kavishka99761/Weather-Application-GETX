## Overview 📙

The Flutter Weather App is a visually captivating and fully functional User Interface for discovering weather conditions. The app allows users to access accurate weather information for their current location as well as cities around the world. It provides real-time weather updates, forecasts, and detailed weather insights.

---

## Dependencies 📦️

* get - Manage states and inject dependencies.
* dio - A powerful HTTP networking package for Dart/Flutter.
* location - Easy access to device location in real-time.
* smooth_page_indicator - Customizable animated page indicators.
* flutter_spinkit - Collection of loading animations.
* cached_network_image - Load and cache network images.
* shimmer - Add shimmer loading effects.
* intl - Date and number formatting.
* flutter_screenutil - Responsive screen adaptation.
* shared_preferences - Persistent local storage.
* flutter_animate - Beautiful Flutter animations.
* carousel_slider - Carousel slider widget.
* flutter_svg - SVG rendering library.
* lottie - Render After Effects animations.
* step_progress_indicator - Step-based progress indicators.

## Features 🌟

* Discover weather at your current location.
* Search weather conditions around the world.
* 24-hour weather forecast.
* Sunrise and sunset times.
* Wind speed and direction.
* Humidity, pressure, UV index, and rain probability.
* Multi-language support (English and Arabic).
* Light and Dark theme support.
* Modern and user-friendly UI.

## Getting Started

### 1. Get Your API Key

Create an account at [WeatherAPI](https://www.weatherapi.com) and obtain your API key.

### 2. Clone the Repository

```bash
git clone https://github.com/Kavishka99761/flutter_weather_app.git
```

### 3. Install Dependencies

```bash
flutter pub get
```

### 4. Add Your API Key

Navigate to:

```text
lib/utils/constants.dart
```

Replace:

```dart
static const mApiKey = 'Your API Key';
```

with your WeatherAPI key.

### 5. Run the Application

```bash
flutter run
```
